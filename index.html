🎯 How to Use This Simulation

  • TouchLED Button: Makes the robot move up one floor (200mm forward)
  • Bumper Button: Makes the robot move down one floor (200mm reverse)
  • Red Color Detection: Click any floor sensor to make it red - robot stops and shows "Arrived"
  • Obstacle Detection: Toggle obstacle to trigger emergency stop with red light
  • Brain Screen: Shows current robot status and floor information
  • Robot Lights: Yellow = arrived at floor, Red = emergency stop, Gray = normal
